## 📝 論文メモテンプレート

以下のテンプレートを使って、読んだ日本語論文を可能な限りまとめていく🚀2025/3/20~


📌 **使い方**  
1. 新しい論文を読んだらこのテンプレートをコピー＆ペースト  
2. 各項目を埋めて簡潔にまとめる  
3. 自分のコメントや考察も忘れずに！  

✨ **ポイント**  
- 「一言でいうと」は、研究のキモを短くまとめる  
- 「新規性」では、他研究との差別化を意識する  
- 「コメント」は自分の考察を入れておくと後で役立つ
